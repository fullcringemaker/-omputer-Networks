Цель работы:
Создать FTP-клиент с возможностью асинхронной передачи команд и динамического отображения состояния файловой системы.


Задание:
Разработать веб-приложение, которое предоставляет пользователю форму для ввода реквизитов доступа к FTP-серверу, включая адрес сервера, логин и пароль. Реализовать консоль для ввода FTP-команд, обеспечивая их асинхронную обработку. Обеспечить асинхронное обновление состояния текущей директории, визуально отличая файлы от папок и отображая права доступа для каждого объекта. Приложение должно автоматически обновлять отображаемую информацию без необходимости ручного обновления и предоставлять интуитивно понятный интерфейс для взаимодействия с файловой системой FTP-сервера.


к данной задаче составь вывод по написанному заданию лабораторной работы, который должно начинаться со слов  "В данной лабораторной работе..."
вывод сделай по коду в контексте задачи, но без упоминания конкретных классов. например, упомяни что и для чего используется  в контексте того, что, исходя из задания, требуется найти или вывести. 
Вывод должен быть в 1 абзац
